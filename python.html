1. Variables

Think of variables same as variables in Simple Maths. The only thing here is we can assign a lot more than just numbers to any word. We assign some value to the variable which gets stored in your machine's memory & can further be changed. Lets say you want to define a variable 'x' and give it a value of 5 :


x = 5


As simple as that! You dont have to specify the data type of the variable value or put any semi-colon at the end of any line like we do in C, C++, Java, etc.


Lets try some more definitions :


one = 1
num = 99999999999999999
num = 3  
# Here I assigned a different value for the variable 'num'. We call it as overwriting.

# By the way the lines in which I'm explaining to you are known as comments.
# They begin with a '#' mark and a space and continue till the end of the line.
# Comments do not execute in a program. They are like ignored by the machine.


Besides integers, we can use many other data types :


# float - decimal numbers

# boolean - True, False.

2. Conditional Statements (if / else / elif)

if else statements are used when you have to check any expression or condition. If the expression given for if results in True, the code inside it executes; if not, the code inside else executes. For example :


if 9 > 4 :
  print('9 is greater than 4')
else :
  print('9 is smaller than 4')

# Here 9>4 is True so, code inside if will be executed.

if 2 > 4 :
  print('2 is greater than 4')
else :
  print('2 is smaller than 4')

# Here 2>4 is True so, code inside else will be executed.


You can use elif statements if you have multiple conditions :


x = 5

if x >= 9 :
  print('x is greater than or equal to 9')
elif x <= 1 :
  print('x is smaller than or equal to 1')
else :
  print('x lies between 1 and 9')

# Here '<=' '>=' are relational operators used to compare values.

3. Looping / Iterations (for / while / while-else)

Looping in python is done in three ways. The for loop, the while loop and the while else loop.


While Looping: While loop needs a statement or an expression which results in either True or False; the code inside the block will be executed if it is True, and the loop terminates if it is False. So, this code will print the number from 1 to 10.


num = 1

while num <= 10 :
  print(num)
  num += 1
# The code inside the loop keeps on executing untill num = 10
# As soon as num = 11, the statement becomes False and the loop terminates.
# Here '+=' is assignment operator. num += 1 is actually num = num + 1


While-else Looping: As soon as while loop terminates or does'nt loop, else block code gets executed. For example :


num = 1

while num <= 10 :
  print(num)
  num += 1
else :
  print('The while loop has finished executing.')


For Looping: For loops are generally used if you want to loop sequentially. For example: traversing a list or string or array etc. The following code will print the same as while code: from 1 to 10.


for num in range(1, 11) :
  print(num)
# Here range(1, 11) generates a sequence of 1-10 numbers. It is a built-in function.
# In range(1 ,11) the upper limit that is 11 here is excluded and 10 is included.

for letter in 'Hello World' :
  print(letter)
# Output : H e l l o W o r l d (one letter each on one line)

4. Operators


Operators as the name suggests perform operations.
Let's look at some of the operators, their symbols and operations / uses :


# Arithmetic Operators : +, -, *, / are the common ones.
# // gives floor value , % gives the remainder , ** denotes exponential.

print(9 // 5) # 1.8
print(9 % 5) # 4
print(3**2) # 9

# Relational operators : used to compare values , returns either True or False.
print(9 > 5) # True
print(9 <= 5) # False
print(4 == 4) # True (single '=' is an assignment operator used to assign values to operands on left side)
print(4 != 4) # False (not equal to)

# Logical operators : Logical operators perform Logical AND, Logical OR and Logical NOT operations.
print(9 > 5 and 1 <= 0) # False , since True and False gives false in Logical and.
print(1 and 0) # False , you can also use 1 and 0 to denote True and False respectively.
print(1 or 0) # True
print(not 9 > 5) # False , since 9>5 is True and 'not' makes it opposite.

# Bitwise operators : Similar to logical operators
# The difference here is the operations performed are bitwise - bit by bit.
# '&' - Bitwise and , '|' - Bitwise or , '~' - Bitwise not
a = 8
b = 5

print(a & b) # 0
print(a | b) # 13
print(~a) # -9

# Assignment operators: Assignment operators are used to assign values to the variables.
# The most common one is '=' which assigns value to the operand on LHS.
a = 2
b = 3

a += b # 5 , it simply means a = a + b
a -= b # 1 , it simply means a = a - b
a *= b # 6 , it simply means a = a * b
a /= b # 0.6666 , it simply means a = a / b

5. Lists / Dictionaries / Tuples / Sets

Lists / Dictionaries / Tuples / Sets are 4 different built-in data structures in Python.
Data structure is a data which is organized & stored in a very specific manner so that necessary edits can be done efficiently on that data.


Ok now you've learnt how to assign some value to a variable. What if you want to assign more than one value to a variable ; say - 1, 2, 3, 4, 5, 6? Or 'Rahul', 'Tushar', 'Himanshu' ? Or 1, 3.8, True, 'Hello', 'World' ?
For that we have different collections / containers in Python. Lets have a look at them one by one :


List : It is simply an ordered collection / container of different objects or items which can be accessed individually. For example lets create a list containing these items : 1, 3.8, True, 'Hello', 'World' .
( Yes ! A list can contain items of different data types or data structure. )


list1 = [1, 3.8, True, 'Hello', 'You are Smart and Beautiful', [2, 4, 6]]
# You need to enclose your items inside '[]' these square brackets to make it a list.


That's it. You created a list that easily. Now what if you have to access one of those items ?

We access them by providing indices of the items. Index is simply a rank of that item from the start. In Python indexing starts from 0. The first item has index 0, second item 1, third 2, likewise nth item has index n-1.


list1[2] # True
list1[0] # 1

# You can also access by providing negative indexing
# -1 for the last item , -2 for second last , -3 for third last and so on...

list1[-1] # [2, 4, 6]
list1[-6] # 1

# You can also add elements at a particular index

list[0] = 0 # [0, 3.8, True, 'Hello', 'You are Smart and Beautiful', [2, 4, 6]]
list[-1] = 'last' # [0, 3.8, True, 'Hello', 'You are Smart and Beautiful', 'last']


Lets Go ! You learnt to create a list and access its elements. But how do you modify it ? How do you remove a specific element or add items in it further ? For that we have list methods. Method is nothing but a way to alter an object specifically; similar to functions. Some of the common list methods are :



Dictionary : It is an ordered collection of items in which the items are in the form of 'Key : Value' pair. Just similar to lists but instead of indexing by numbers we here have 'key'. The key is the index pointing to the value.
Lets look at an example :


dict1 = {'one' : 1, 'two' : 2, 'three' : 3, 7 : 'Dhoni', 5.5 : 'five point five', 'list1' : [2, 4, 6]}
# We have to provide two things seperated by a ':' for each item.
# The left one is the key and the right one is the value for that key.
# Enclose whole your data inside '{}' curly brackets.
# Each value can be accessed with the help of its respective key.

dict1['one'] # 1
dict1[7] # Dhoni
dict1[5.5] # 'five point five'
dict1['list1'] # [2, 4, 6]

# The value can also be obtained with the help of '.get()' method

dict1.get(7) # 'Dhoni'

# You can also add elements for a particular key

dict1['new'] = 'Hello world'



For detailed explaination of Dictionary methods, visit here


Tuples : Shortly & simply, Tuples are nothing but immutable lists. It means that they are ordered but you cannot change, add, remove any data inside it.
Lets create one :


tuple1 = ("abc", 34, True, 4.123, "male")
# Tuple brackets : '()'

tuple1[2] = False # TypeError : 'tuple' object does not support item assignment


You can access the elements just like lists.


But remember : if your tuple has only one element, give a comma after it.


tuple1 = ("abc",)

tuple1 = ("abc") # Not a tuple

# These tuples are often used for iterations in loops.


Sets : Are again a collection of data but this time they are unordered and does'nt have duplicate items.
We can't access the set items with indexing due to their unorderedness which means everytime you use them they appear in different order. They are written in curly brackets '{}' .


set1 = {"abc", 34, True, 40, "male"}

6. Looping through data containers



Now here comes the most important use of data containers - Iterations! Most of the times we need individual elements of that list one by one to perform some tasks or operations. And if you ask me from my experience pov, for loops are used almost every time. So let us see how its done :


data = [7, 13, 3, 2934, 44, 123, 0]

for num in data :
  if num % 2 == 0 :
    print(num)

# This is a short code for printing even numbers.
# Carefully have a look at the syntax.
# After 'for' specify the variable name you are going to use throughout the loop for that individual item.
# 'in _data_container_' assigns a single item to the variable sequentially for each loop.

# But there's a twist. You can only use that variable in that loop. You cannot use it outside that loop.
# Similar to local variables.


We saw how to iterate through a list. Now lets get to Dictionaries.
We know that a dictionary contains items in the form of key-value pairs. So its obvious that we will have to iterate through the key value pairs. Lets look at our previous example :


dict1 = {'one' : 1, 'two' : 2, 'three' : 3, 7 : 'Dhoni', 5.5 : 'five point five', 'list1' : [2, 4, 6]}

for key, value in dict1.items :
  print(key, value)

# one, 1
# two, 2
# three, 3
# 7, Dhoni
# 5.5, five point five
# list1, [2, 4, 6]


We did name the two parameters as key and value, but we can also use any other word or name :


dict2 = {'Tushar' : 20, 'Pratik' : 18, 'Himanshu' : 21, 'Geeta' : 20}

for name, age in dict2.items :
  print(f'{name} : {age}')

# Tushar : 20
# Pratik : 18
# Himanshu : 21
# Geeta : 20

# the code used above - f'---' to print the result is known as f string in Python.


Bonus tip : You can also iterate through strings for each character in it.


name = 'Zlatan'

for letter in name :
  print(letter)

# Z
# l
# a
# t
# a
# n
